# PixelStream – Real-Time Media Streaming Application

PixelStream is a real-time media streaming application developed using Python and socket programming. The system enables seamless communication between clients and servers for media transmission.

## Features

- Real-time audio and video streaming
- Client-server architecture
- Socket-based communication
- Multi-client support
- Media management
- User-friendly interface
- Low-latency data transmission

## Technologies Used

- Python
- Socket Programming
- OpenCV
- PyAudio
- CustomTkinter
- Threading

## System Architecture

- Server Module
- Client Module
- Video Streaming Module
- Audio Streaming Module
- Chat Module

## Installation

1. Clone the repository:

   git clone https://github.com/ompatil610/PixelStream.git

2. Navigate to the project directory:

   cd PixelStream

3. Create and activate a virtual environment:

   python -m venv venv

   Windows:
   venv\Scripts\activate

4. Install dependencies:

   pip install -r requirements.txt

5. Start the server:

   python server.py

6. Start the client:

   python client.py
